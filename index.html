102 Etiquetas HTML

# Etiqueta Descripción Ejemplo de Uso Comentarios / Elemento Adicional
1 <a> Define un hipervínculo. <a href="https://ejemplo.com">Visítanos</a> Permite navegar entre páginas.
2 <abbr> Define una abreviatura con su significado. <abbr title="HyperText Markup Language">HTML</abbr> Muestra el significado al pasar el cursor.
3 <address> Define información de contacto. <address>contacto@empresa.com</address> Usado frecuentemente en pie de página.
4 <area> Define áreas dentro de una imagen (mapa). <area shape="rect" coords="34,44,270,350" href="link.html"> Debe usarse dentro de <map>.
5 <article> Representa contenido independiente y autocontenible. <article>Noticia importante</article> Ideal para blogs y noticias.
6 <aside> Define contenido relacionado pero independiente del principal. <aside>Publicidad</aside> Usado en barras laterales.
7 <audio> Inserta contenido de audio. <audio controls src="audio.mp3"></audio> Soporta múltiples formatos de audio.
8 <b> Texto en negrita sin importancia adicional. <b>Importante</b> Usar <strong> para énfasis semántico.
9 <base> Establece la URL base para enlaces relativos. <base href="https://ejemplo.com/"> Debe colocarse dentro del <head>.
10 <bdi> Aisla una parte del texto que puede tener dirección diferente. <bdi>اسم المستخدم</bdi> Útil para contenido en idiomas bidireccionales.
11 <bdo> Reescribe la dirección del texto (de derecha a izquierda o viceversa). <bdo dir="rtl">Texto al revés</bdo> Útil para idiomas con escritura RTL como árabe o hebreo.
12 <blockquote> Define una cita textual de otra fuente. <blockquote cite="https://ejemplo.com">Texto citado</blockquote> Visualmente se sangra del resto del contenido.
13 <body> Contiene todo el contenido visible de la página. <body>Contenido principal</body> Solo debe haber una etiqueta <body> por página.
14 <br> Inserta un salto de línea. Primera línea<br>Segunda línea Es una etiqueta vacía (sin cierre).
15 <button> Define un botón clickeable. <button type="button">Haz clic</button> Usado en formularios e interacciones.
16 <canvas> Área de dibujo dinámico con JavaScript. <canvas id="miCanvas" width="200" height="100"></canvas> Requiere JavaScript para mostrar contenido.
17 <caption> Define el título de una tabla. <caption>Lista de productos</caption> Debe ir inmediatamente después de la apertura de <table>.
18 <cite> Usado para citar el título de una obra. <cite>El Quijote</cite> Suele mostrarse en cursiva por defecto.
19 <code> Representa código de programación. <code>console.log("Hola")</code> Usado para destacar fragmentos de código.
20 <col> Define propiedades de una columna en una tabla. <col style="background-color: #f2f2f2;"> Debe estar dentro de <colgroup>.
21 <colgroup> Agrupa una o más columnas en una tabla. <colgroup>
  <col span="2" style="background-color:lightgrey;">
</colgroup> 		Usado para aplicar estilos a columnas.
22 <data> Asocia un valor legible con un valor de máquina. <data value="12345">Artículo 12345</data> Útil para procesamiento con JavaScript.
23 <datalist> Contiene opciones predefinidas para un input. <input list="navegadores">
<datalist id="navegadores">
  <option value="Chrome">
  <option value="Firefox">
</datalist> 		Mejora la experiencia de autocompletado.
24 <dd> Define la descripción de un término en una lista de definiciones. <dd>Lenguaje de marcado para la web.</dd> Usado junto a <dt> y <dl>.
25 <del> Texto eliminado o tachado. <del>Texto antiguo</del> Puede usarse para mostrar cambios en revisiones.
26 <details> Crea un widget que el usuario puede expandir o colapsar. <details>
  <summary>Más info</summary>
  Contenido adicional
</details> 		Ideal para FAQs o contenido adicional oculto.
27 <dfn> Marca un término definido por primera vez. <dfn>HTML</dfn> es un lenguaje de marcado. Puede mostrar un estilo especial o tooltip.
28 <dialog> Define una ventana de diálogo (modal). <dialog open>Este es un diálogo</dialog> Requiere JS para abrir/cerrar dinámicamente.
29 <div> Contenedor genérico para contenido. <div class="caja">Contenido</div> Muy usado para diseño con CSS.
30 <dl> Lista de definiciones (términos + descripciones). <dl>
  <dt>HTML</dt>
  <dd>Lenguaje de marcado</dd>
</dl> 		Se usa con <dt> y <dd>.
31 <dt> Define un término en una lista de definiciones. <dt>CSS</dt> Se usa con <dd> y <dl>.
32 <em> Define énfasis en el texto (cursiva por defecto). <em>Importante</em> Usado para dar énfasis semántico al texto.
33 <embed> Incorpora contenido externo (como multimedia o PDF). <embed src="archivo.pdf" type="application/pdf"> Similar a <object> pero más simple.
34 <fieldset> Agrupa elementos relacionados en un formulario. <fieldset>
  <legend>Datos</legend>
  <input type="text">
</fieldset> 		Usado para organización visual y lógica en formularios.
35 <figcaption> Proporciona un título o leyenda a un <figure>. <figure>
  <img src="img.jpg">
  <figcaption>Descripción</figcaption>
</figure> 		Debe estar dentro de una etiqueta <figure>.
36 <figure> Contenedor para ilustraciones, imágenes o diagramas. <figure><img src="foto.jpg"></figure> Puede tener una <figcaption> asociada.
37 <footer> Define el pie de una sección o página. <footer>Derechos reservados</footer> Contiene información como autor, links, copyright.
38 <form> Contenedor para elementos de formulario. <form action="/submit">
  <input type="text">
</form> 		Usado para enviar datos al servidor.
39 <h1> a <h6> Encabezados jerárquicos de nivel 1 a 6. <h1>Título principal</h1> <h1> es el más importante, <h6> el menos.
40 <head> Contiene metadatos del documento. <head>
  <title>Mi sitio</title>
</head> 		No visible en el cuerpo de la página.
41 <header> Define un encabezado para una sección o página. <header>Bienvenido a mi web</header> Normalmente contiene logo, navegación, título.
42 <hr> Inserta una línea horizontal divisoria. <hr> Útil para separar secciones.
43 <html> Elemento raíz de un documento HTML. <html lang="es">...</html> Debe contener todos los elementos de la página.
44 <i> Texto en cursiva sin énfasis semántico. <i>Palabra extranjera</i> Para diferenciación visual, no semántica.
45 <iframe> Inserta otra página web dentro de la actual. <iframe src="https://ejemplo.com"></iframe> Usado para mapas, videos, widgets.
46 <img> Inserta una imagen en la página. <img src="foto.jpg" alt="Descripción"> Debe tener atributo alt por accesibilidad.
47 <input> Campo de entrada en formularios. <input type="text"> Muy versátil (text, checkbox, radio, etc.).
48 <ins> Texto insertado (nuevo contenido). <ins>Texto añadido</ins> Puede mostrarse subrayado.
49 <kbd> Representa la entrada del teclado. Presiona <kbd>Ctrl</kbd> + <kbd>C</kbd> Muy útil para tutoriales o instrucciones.
50 <label> Etiqueta para un <input>. <label for="nombre">Nombre:</label><input id="nombre"> Mejora accesibilidad en formularios.
51 <legend> Define un título o leyenda para un <fieldset>. <fieldset>
  <legend>Información personal</legend>
  <input type="text">
</fieldset> 		Proporciona contexto a los campos del formulario.
52 <li> Elemento de una lista. <ul><li>Elemento 1</li></ul> Funciona dentro de <ul>, <ol> o <menu>.
53 <link> Vincula recursos externos como hojas de estilo. <link rel="stylesheet" href="estilos.css"> Va dentro de <head>.
54 <main> Define el contenido principal del documento. <main>Contenido principal aquí</main> Debe haber solo un <main> por documento.
55 <map> Define un mapa de imagen con áreas clickeables. <img src="imagen.jpg" usemap="#mapa">
<map name="mapa">
  <area shape="rect" coords="34,44,270,350" href="enlace.html">
</map> 		Usado con <area>.
56 <mark> Resalta texto importante o relevante. <mark>Texto resaltado</mark> Suele mostrarse con fondo amarillo.
57 <meta> Define metadatos del documento. <meta charset="UTF-8"> Va dentro del <head>, no tiene cierre.
58 <meter> Representa una medida escalar dentro de un rango. <meter value="0.6">60%</meter> Ideal para puntuaciones, porcentajes, etc.
59 <nav> Define una sección de navegación. <nav>
  <a href="#">Inicio</a>
  <a href="#">Contacto</a>
</nav> 		Usado para menús de navegación.
60 <noscript> Contenido alternativo si JavaScript está deshabilitado. <noscript>Activa JavaScript para ver esta página</noscript> Se muestra solo si el navegador no soporta JavaScript.
61 <object> Incorpora contenido multimedia o interactivo. <object data="archivo.pdf" type="application/pdf"></object> Más flexible que <embed>, pero más complejo.
62 <ol> Lista ordenada (numerada). <ol><li>Uno</li></ol> Los ítems se muestran con números.
63 <optgroup> Agrupa opciones en un <select>. <select>
  <optgroup label="Frutas">
    <option>Manzana</option>
  </optgroup>
</select> 		Mejora la organización de listas desplegables.
64 <option> Opción dentro de un menú desplegable. <option value="1">Uno</option> Usado dentro de <select> o <datalist>.
65 <output> Muestra el resultado de un cálculo. <output>Resultado</output> Puede interactuar con JavaScript o formularios.
66 <p> Define un párrafo de texto. <p>Este es un párrafo.</p> Elemento básico de contenido textual.
67 <param> Define parámetros para plugins o <object>. <param name="autoplay" value="true"> No tiene efecto fuera de <object>.
68 <picture> Contenedor para mostrar imágenes según condiciones. <picture>
  <source srcset="img.webp" type="image/webp">
  <img src="img.jpg" alt="Imagen">
</picture> 		Ideal para imágenes responsivas.
69 <pre> Texto preformateado (con espacios y saltos). <pre>
Línea 1
Línea 2
</pre> 		Mantiene el formato original del texto.
70 <progress> Barra de progreso visual. <progress value="70" max="100">70%</progress> Ideal para mostrar avances en tareas.
71 <q> Define una cita corta en línea. <q>Esto es una cita corta</q> El navegador agrega comillas automáticamente.
72 <rp> Texto alternativo para navegadores que no soportan ruby. <rp>(</rp><rt>lectura</rt><rp>)</rp> Usado con <ruby> y <rt>.
73 <rt> Proporciona la pronunciación en ruby annotations. <ruby>漢<rt>kan</rt></ruby> Especialmente útil para caracteres asiáticos.
74 <ruby> Contenedor para anotaciones fonéticas (ruby). <ruby>漢<rt>kan</rt></ruby> Usado en textos japoneses y chinos.
75 <s> Muestra texto como no válido o tachado. <s>Precio antiguo</s> Similar a <del> pero sin semántica de eliminación.
76 <samp> Representa una salida de sistema o programa. <samp>Error 404</samp> Usado para simular salidas de comandos o scripts.
77 <script> Incorpora o enlaza código JavaScript. <script src="app.js"></script> Debe ir al final del <body> o en <head>.
78 <section> Define una sección temática del documento. <section><h2>Noticias</h2></section> Ayuda a estructurar el contenido semánticamente.
79 <select> Crea un menú desplegable. <select>
  <option>Opción 1</option>
</select> 		Se usa en formularios.
80 <small> Muestra texto en tamaño más pequeño. <small>Nota al pie</small> Usado para aclaraciones o advertencias.
81 <source> Especifica múltiples recursos para medios (audio, video). <video controls>
  <source src="video.mp4" type="video/mp4">
</video> 		Debe ir dentro de <audio> o <video>.
82 <span> Contenedor en línea genérico. <span style="color:red">Texto rojo</span> Usado para aplicar estilos o manipular con JS.
83 <strong> Define texto importante (negrita semántica). <strong>¡Importante!</strong> Transmitido a lectores de pantalla como énfasis.
84 <style> Define estilos CSS dentro del HTML. <style>
  p { color: blue; }
</style> 		Debe ir dentro de <head>.
85 <sub> Texto en subíndice. H<sub>2</sub>O Usado en fórmulas químicas, matemáticas, etc.
86 <summary> Título visible de un elemento <details>. <details>
  <summary>Ver más</summary>
  Contenido oculto
</details> 		Se muestra como elemento desplegable.
87 <sup> Texto en superíndice. x<sup>2</sup> Útil para exponentes y referencias.
88 <svg> Contenedor para gráficos vectoriales escalables. <svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="red" />
</svg> 		Ideal para ilustraciones e iconografía.
89 <table> Define una tabla de datos. <table>
  <tr><td>Dato</td></tr>
</table> 		Se compone de <tr>, <th> y <td>.
90 <tbody> Agrupa el cuerpo de una tabla. <table>
  <tbody>
    <tr><td>Fila</td></tr>
  </tbody>
</table> 		Separa visual y semánticamente las secciones de la tabla.
91 <tfoot> Agrupa el pie de una tabla. <table>
  <tfoot>
    <tr><td>Total</td></tr>
  </tfoot>
</table> 		Se usa para sumar datos o notas finales.
92 <th> Define una celda de encabezado en una tabla. <table>
  <tr><th>Nombre</th></tr>
</table> 		Texto en negrita y centrado por defecto.
93 <thead> Agrupa el encabezado de una tabla. <table>
  <thead>
    <tr><th>Columna 1</th></tr>
  </thead>
</table> 		Separa el encabezado para facilitar estilos y accesibilidad.
94 <time> Representa una fecha y/o hora. <time datetime="2025-09-16">16 de septiembre de 2025</time> Útil para máquinas y lectores de pantalla.
95 <title> Define el título de la página que aparece en la pestaña del navegador. <title>Mi página web</title> Debe ir dentro del <head>.
96 <tr> Define una fila en una tabla. <table>
  <tr>
    <td>Celda 1</td>
  </tr>
</table> 		Contiene celdas <td> o encabezados <th>.
97 <track> Define pistas de texto para medios (subtítulos, descripciones). <track kind="subtitles" src="subtitulos.vtt" srclang="es" label="Español"> Debe ir dentro de <audio> o <video>.
98 <u> Subraya texto. <u>Texto subrayado</u> Ideal para enfatizar sin cambiar el significado.
99 <ul> Lista desordenada (con viñetas). <ul>
  <li>Elemento 1</li>
</ul> 		Usado para listas sin orden específico.
100 <var> Define una variable en una expresión matemática o de programación. <var>x</var> + <var>y</var> = <var>z</var> Estilizado en cursiva por defecto.
101 <video> Incorpora contenido de video. <video controls>
  <source src="video.mp4" type="video/mp4">
</video> 		Permite control de reproducción integrado.
102 <wbr> Indica un posible punto de ruptura de línea. palabralarga No tiene cierre y ayuda a mejorar el diseño responsive.
